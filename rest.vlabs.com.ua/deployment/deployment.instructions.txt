-----------------
Collaborator Repository Structure

1) mkdir ~/vlabs.com.ua
2) cd ~/vlabs.com.ua
2) unzip ~/git/vlabs.com.ua/vl.vlabs.com.ua/misc/blender_server_side_export/BLENDER_EXPORT.zip -d .
2) mkdir -p ~/vlabs.com.ua/collaborators/projects
-----------------
Collaborator Repository Server

1) cd ~/vlabs.com.ua/collaborators/projects
2) cp ~/git/vlabs.com.ua/vl.vlabs.com.ua/misc/collaborators-repository-server/collaborator-repository-server.js .
3) npm install express cors express-basic-auth
4) node collaborator-repository-server.js
-----------------
1) Create initial database structure
mysql -uroot -p < init-db.sql

2) Insert initial data records
mysql -uvlabs -p < init-data.sql

-----------------
VLabs server by default is running on 8090 port (`server.port` in application.properties)
